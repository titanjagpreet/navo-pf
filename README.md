# Navodita Singh - Academic Portfolio

This is the personal academic portfolio website for Navodita Singh, a PhD Research Scholar at IISER Bhopal.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Deployment:** Vercel / GitHub Pages

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
